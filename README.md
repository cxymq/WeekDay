# WeekDay

- [x] 包含日期表和滑动视图
- [ ] 记录浏览位置

### 分支
**master : **初始化两个 tableView，进行重复使用。  
**scroll-location : **基于 master 分支，记录每个日期的浏览位置，返回上一级后无效。  
**scroll-all-tables : **日期与 tableView 一一对应，效果看起来比其他分支好。
